# 🏥 SmartHealth care – Full Stack Project

## 📘 Abstract

The **SmartHealth care** is a full-stack web application developed to automate and manage the daily operations of a hospital.  
The system provides an efficient way to handle patient records, doctor information, appointment scheduling, billing, and medical records through a centralized digital platform.

This project reduces paperwork, minimizes human errors, improves data security, and enhances overall hospital efficiency by providing real-time access to information.

---

## 🎯 Objectives

- To automate hospital administrative tasks
- To manage patient and doctor data efficiently
- To provide secure and role-based access
- To improve hospital workflow and productivity
- To maintain accurate medical records digitally

---

## 🛠 Technologies Used

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Spring Security
- RESTful APIs
- MySQL Database

### Frontend
- React.js
- HTML
- CSS
- JavaScript

---

## ✨ Features

- User authentication and authorization
- Role-based access (Admin, Doctor, Patient)
- Patient registration and management
- Doctor management and specialization tracking
- Appointment scheduling and management
- Electronic Medical Records (EMR)
- Billing and payment management
- Secure data storage

---

## 🏗 System Architecture

The system follows a **client-server architecture**:
- Frontend communicates with backend using REST APIs
- Backend processes requests and interacts with the database
- Database stores hospital-related information securely

---

## 🔐 Security

- Secure login system
- Password encryption
- Role-based access control
- Secure REST endpoints

---

## 🗄 Database

- Default database: **MySQL**
- Supports migration to **H2**
- Stores patient records, doctor details, appointments, and billing data
